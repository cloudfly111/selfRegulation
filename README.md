# 自我狀態管理 App
> 目的：主要是由 App 彙整生活中工作、健康指標與壓力相關狀態以管理自身身體狀態
- #### App 適用 Android 版本：Android 9 以上

### build 環境：
- IDE：Android Studio Chipmunk | 2021.2.1 Patch 1
- Kotlin version：1.7.10
### 架構：MVVM
### 預計使用的套件/資料庫
- 本地資料庫：Room
- 遠端資料庫：FireBase RealtimeDatabase 
- API 串接：Retrofit
- 異步處理：Coroutines
